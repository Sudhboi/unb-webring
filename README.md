# unb-webring
<img alt="unb-webring icon" src="assets/favicon.svg" width="220" />
A webring for students and alumni at the University of New Brunswick.

## Join the webring
To add your site, fork this repo and open [javascript/sites.js](javascript/sites.js). Then add your information to the bottom of the `sites` array:

```js
{
  name: "First Last",
  campus: "F", // or "SJ"
	year: "20XX",
	website: "https://your-site.com"
}
```

Commit your change in your fork and open a pull request.
